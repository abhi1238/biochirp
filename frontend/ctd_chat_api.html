CTD Documentation

Data Source

Source files are Comparative Toxicogenomics Database (CTD) downloads from https://ctdbase.org/downloads/, processed as CTD_*.csv inputs in the preprocessing notebook.

Raw Files Used

  • CTD_chem_gene_ixns.csv — chemical ↔ gene associations
  • CTD_curated_chemicals_diseases.csv — chemical ↔ disease associations
  • CTD_curated_genes_diseases.csv — gene ↔ disease associations
  • CTD_genes_pathways.csv — gene ↔ pathway associations
  • CTD_diseases_pathways.csv — disease ↔ pathway associations
  • CTD_chemicals.csv — chemical master
  • CTD_genes.csv — gene master
  • CTD_diseases.csv — disease master
  • CTD_pathways.csv — pathway master

Processing Summary

  • Standardize columns and normalize text.
  • Filter chemical–gene rows to Homo sapiens.
  • Remove ID prefixes (e.g., MESH:, REACT:, KEGG:).
  • Drop missing identifiers and deduplicate.

Derived Tables (Parquet)

  • chemical_master_table.parquet: drug_id, drug_name
  • gene_master_table.parquet: gene_id, gene_name
  • disease_master_table.parquet: disease_id, disease_name
  • pathway_master_table.parquet: pathway_id, pathway_name
  • chemical_gene_association.parquet: drug_id, gene_id
  • chemical_disease_association.parquet: drug_id, disease_id
  • gene_disease_association.parquet: gene_id, disease_id
  • gene_pathway_association.parquet: gene_id, pathway_id
  • disease_pathway_association.parquet: disease_id, pathway_id

Supported Query Types

  • Chemical ↔ gene, chemical ↔ disease
  • Gene ↔ disease, gene ↔ pathway
  • Disease ↔ pathway
  • Lookup of master tables (chemical, gene, disease, pathway)

Sample Questions

  • Which drugs are associated with tuberculosis?
  • Which genes are associated with asthma?
  • Which pathways are associated with Alzheimer’s disease?
  • Which genes are associated with benzene?
  • Which diseases are associated with lead?
  • Which pathways are associated with TP53?

Special Notes

  • CTD data represent curated associations, not clinical recommendations.
  • Use specific entity names for best precision.
